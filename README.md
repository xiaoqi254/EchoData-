## 登录问题

### 忘记密码怎么办？
1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件

### 账号被锁定怎么办？
请联系客服解锁，提供账号信息即可。

## 检测问题

### 为什么检测失败？
常见原因：
- 号码格式不正确
- 余额不足
- 文件格式错误

### 支持哪些号码格式？
支持：手机号、固话、国际号码。
